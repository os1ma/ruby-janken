※ 記載しているコマンドは、全てプロジェクトのルートディレクトリで実行することを想定しています

## pre-commit の設定

```shell
ln -s ../../bin/pre-commit .git/hooks/pre-commit
```

## 実行

```shell
ruby ./lib/janken_cli.rb
```

## テスト

```shell
./bin/test.sh
```
